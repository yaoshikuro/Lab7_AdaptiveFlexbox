## Команды Flexbox

### 1. display: flex
Включает Flexbox на контейнере. Элементы внутри встают в ряд.

**display: flex;**

---

### 2. flex-direction

**flex-direction: row;** - элементы располагаются слева направо

**flex-direction: column;** - элементы располагаются сверху вниз

**flex-direction: row-reverse;** - элементы располагаются справа налево

**flex-direction: column-reverse;** - элементы располагаются снизу вверх

---

### 3. align-items

**align-items: stretch;** - растягивает элементы на всю высоту

**align-items: flex-start;** - прижимает элементы к верхнему краю

**align-items: flex-end;** - прижимает элементы к нижнему краю

**align-items: center;** - выравнивает элементы по центру

**align-items: baseline;** - выравнивает элементы по тексту

---

### 4. flex-wrap

**flex-wrap: nowrap;** - все элементы в одной строке (не переносятся)

**flex-wrap: wrap;** - элементы переносятся на новую строку

**flex-wrap: wrap-reverse;** - элементы переносятся на новую строку в обратном порядке

---

### 5. gap

**gap: 10px;** - отступ между элементами 10 пикселей

**gap: 20px;** - отступ между элементами 20 пикселей

---

### 6. flex-grow

**flex-grow: 1;** - элемент растягивается и занимает свободное место

**flex-grow: 2;** - элемент растягивается в 2 раза больше других

---

### 7. justify-content

**justify-content: flex-start;** - элементы прижаты к началу

**justify-content: flex-end;** - элементы прижаты к концу

**justify-content: center;** - элементы по центру

**justify-content: space-between;** - первый в начале, последний в конце, остальные между

**justify-content: space-around;** - равные отступы вокруг элементов

**justify-content: space-evenly;** - равные отступы между элементами и по краям

---

## Примеры

### Пример 1. Навигационное меню
**display: flex;**
**justify-content: space-around;**

### Пример 2. Карточки товаров
**display: flex;**
**flex-wrap: wrap;**
**gap: 20px;**

### Пример 3. Центрирование
**display: flex;**
**justify-content: center;**
**align-items: center;**
